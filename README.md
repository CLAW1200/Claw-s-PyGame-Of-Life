# PyGame-Of-Life
Conway's Game Of Life in PyGame

Controls:

- Left Click: Place Cell

- Right Click: Remove Cell

- Space: Pause/Play

- R: Randomize Cells

- C: Clear Cells

- E: Toggle Place Cell

- W: Toggle Remove Cell

- P: Randomize Colours

- Up Arrow/MScroll: Speed Up

- Down Arrow/MScroll: Slow Down

- Right Arrow: Next Generation

- H: Toggle UI

- Esc: Quit
